eHotel App

Application Java + PostgreSQL pour rechercher une chambre, faire une réservation et créer une location.

Recherche de chambres

Recherchez des chambres disponibles selon vos critères. Tous les champs sont optionnels.

Créer une réservation

Sélectionnez un client et une chambre pour créer une réservation.

Créer une location

Choisissez le type de location : directe ou conversion d'une réservation existante.

Type de location

Convertir une réservation en location

Données

Cliquez pour charger et afficher les listes de clients, employés, hôtels, chambres et réservations.
