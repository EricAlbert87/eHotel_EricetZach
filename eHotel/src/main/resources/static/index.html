Application eHotel

Application Java + PostgreSQL pour rechercher une chambre, faire une réservation et créer une location.

Trouvez les meilleures chambres selon vos critères de recherche.

Réservez une chambre pour un client. La réservation ne sera pas confirmée tant que le client n'aura pas effectué le check-in.

Créez une location directe ou convertissez une réservation existante en location avec check-in.
