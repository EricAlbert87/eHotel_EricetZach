# eHotel Application

## Start the Application (Local)

1. Go to the project folder:

```bash
cd /path/to/eHotel
```

2. Install all requirements and initialize the database:

```bash
make setup
```

3. Build the jar:

```bash
make build
```

4. Run the application:

```bash
make run
```

5. Open the app:

http://localhost:8080
